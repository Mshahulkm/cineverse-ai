# CineVerse AI

A premium cinematic movie discovery platform with AI-powered recommendations, immersive browsing, and intelligent search.

## Features

- **AI-Powered Recommendations** — Smart movie suggestions based on mood, time, and preferences
- **Cinematic UI** — Dark theme with neon gradients, glassmorphism, and smooth animations
- **Intelligent Search** — Natural language search with semantic matching
- **Movie Details** — Comprehensive movie pages with trailers, ratings, and streaming info
- **Category Discovery** — Browse by genre, language, country, director, and mood
- **Responsive Design** — Optimized for desktop, tablet, and mobile

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Data Sources

Movie data is sourced from:
- MSone Malayalam Subtitles (malayalamsubtitles.org)
- Malayalam Subtitles (malayalamsubtitles.in)
- Movie Mirror Subtitles (moviemirrorsubtitles.com)
